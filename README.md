# Cuppa Login

Interactive React login screen built from a Figma design for the **Cuppa** coffee app.

## Features

- Pixel-faithful recreation of the Figma design
- Interactive Name & Phone Number inputs (phone enforces exactly 10 digits)
- Login button with validation and success toast
- OAuth modal flow for **Google**, **Facebook**, and **GitHub** sign-in
  - Provider-branded slide-up sheet
  - Animated authorizing spinner
  - Animated success state
- Bemdayni Demo font loaded locally from dafont.com
- Built with React 18, Tailwind CSS v4, Motion, and Sonner

## Stack

- React 18
- Vite
- Tailwind CSS v4
- Motion (Framer Motion)
- Sonner (toasts)
- Radix UI

## Getting Started

```bash
npm install
npm run dev
```
